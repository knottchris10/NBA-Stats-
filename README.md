# NBA Stats Analysis

## Overview
This project analyzes NBA player and team statistics using Python and data analysis libraries. The goal is to identify performance trends, compare players, and uncover insights from historical NBA data.

## Objectives
- Analyze player performance metrics
- Compare teams and players
- Identify statistical trends
- Visualize key NBA statistics

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook / Google Colab

## Dataset
The dataset contains NBA player statistics including:

- Points Per Game (PPG)
- Rebounds
- Assists
- Field Goal Percentage
- Minutes Played
- Team Information
